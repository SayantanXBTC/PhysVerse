## ðŸ“Š Real-Time Data Charts

### How to Access:

#### In Simulation Editor:
1. Open any simulation in the editor
2. Look for the **"Show Chart"** button in the top toolbar
3. Click to toggle the chart on/off

#### In Preview Mode:
1. Go to "Try PhysVerse" preview page
2. Look for the **"Show Data Chart"** button below Play/Pause
3. Click to toggle the chart on/off

### What You'll See:
- Real-time line chart showing:
  - **Velocity** (red line) - Speed of objects in m/s
  - **Energy** (orange line) - Total energy in Joules
  - **Position** (green line) - Position in meters
- Chart updates live as simulation runs
- Last 150 data points displayed
- Time on X-axis, values on Y-axis

### Features:
- **Responsive**: Adapts to screen size
- **Interactive**: Hover over lines to see exact values
- **Performance**: Optimized for 60 FPS
- **Toggleable**: Hide when you want full-screen simulation
