## ðŸ†˜ Getting Help

### Common Issues

**Performance drops?**
- Reduce trail length (e.g., 2000 â†’ 500)
- Lower particle count
- Reduce geometry segments (32 â†’ 16)

**Memory leaks?**
- Check cleanup() method
- Ensure all geometries disposed
- Ensure all materials disposed

**Colors look wrong?**
- Adjust HSL values in code
- H = hue (0-1): 0=red, 0.3=green, 0.6=blue
- S = saturation (0-1): 0=gray, 1=vibrant
- L = lightness (0-1): 0=black, 0.5=normal, 1=white

**Trails don't appear?**
- Check points array length > 1
- Verify updateTrail() is called
- Check material opacity > 0
